== Description ==
Easily create a stunning, high-performing website with just a few clicks with Noor theme.
It runs and adds its enhancements only if the Noor theme version 6.0 and up is installed and active.

== Installation ==

1. Upload `noor-starter-templates.zip` to the `/wp-content/plugins/` directory and extract.
2. Activate the plugin by going to **Plugins** page in WordPress admin and clicking on **Activate** link.
